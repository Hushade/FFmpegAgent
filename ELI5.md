# FFmpegAgent 導入方法

このガイドはパソコンに詳しくない方向けに、FFmpeg のインストールから FFmpegAgent の実行までをステップごとに説明します。

## 導入手順

### 手順①：FFmpeg をインストール（最初の 1 回だけ）

#### FFmpegがインストール済みか確認する
- コマンドプロンプトを開いて、下を実行してください:
  ```cmd
  ffmpeg -version
  ```
- バージョン情報が表示されたら → スキップして「手順②」へ進んでください。
- 「ffmpeg が見つかりません」と表示されたら → 下のインストール手順を実行します。

#### FFmpegがなければインストールする（Winget を使います）
- コマンドプロンプトを管理者として起動します。
- 次を実行:
  ```cmd
  winget install ffmpeg
  ```
- インストール完了後、コマンドプロンプトを再起動して `ffmpeg -version` で確認します。

### 手順②：FFmpegAgent をダブルクリック

- エクスプローラーで `FFmpegAgent` をダブルクリックして実行します。
- コマンドプロンプトのウィンドウが開きます。

### 手順③：切り抜くファイルをドラッグアンドドロップ

- 画面に「入力ファイルのパスを入力してください」と表示されます。
- エクスプローラーで切り取りたい動画ファイル（例: `myGOATクリップ`）をコマンドプロンプトのウィンドウにドラッグアンドドロップします。
- ファイルパスが自動入力されるので、Enter キーを押します。

### 手順④：開始時間を入力

- 「動画の切り取りの開始時間(MM:SS)>」と表示されます。
- 切り取りを始めたい時刻を入力します。
  - 例: `01:23`（1 分 23 秒から）、`03:10.500`（3 分 10.5 秒まで）
- Enter キーを押します。

### 手順⑤：終了時間を入力

- 「動画の切り取りの終了時間(MM:SS)>」と表示されます。
- 切り取りを終わりたい時刻を入力します（開始時間より後の時刻）。
- Enter キーを押します。

### 手順⑥：作業終了まで待つ

- FFmpeg が動画を処理します。ウィンドウに処理の進行状況が表示されます。
- 処理が完了するまで待ちます（ファイルサイズや動画の長さによって数秒～数分かかります）。
- 完了後、コマンドプロンプトウィンドウが自動で閉じます（または Enter キーを押します）。

### 手順⑦：出力ファイルを確認

- 元のファイルと同じフォルダに `元ファイル名-Clip` のファイルが作成されています。
  - 例: `video` → `video-Clip`
- 新しいファイルを再生して、正しく切り抜かれているか確認します。

## よくあるトラブル
- インストールしたのに「ffmpeg が見つかりません」と表示されたら → コマンドプロンプトを再起動し、`ffmpeg -version` を再度確認。
- ファイルが見つからないと言われたら → ファイルパスをコピー＆貼り付けして試す。
- `File '~~~-Clip.mp4' already exists. Overwrite? [y/N]`と出て先に進まない → クリップされた成果物と同じ名前のファイルが既にあるときにでます。「上書き保存しますがよろしいですか？」という質問です。続行して上書き保存なら`y`、キャンセルするなら`n`を打ってエンターを押下してください。

補足:
- 設定の変更は [`Config.ini`](Config.ini) を編集してください。
- 高精度な切り取りが必要な場合は [`README.md`](ELI5.md) を参照してください。
